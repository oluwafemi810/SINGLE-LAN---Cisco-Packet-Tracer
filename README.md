# LAN Configuration Project – Cisco Packet Tracer

This project demonstrates a Local Area Network (LAN) setup using **Cisco Packet Tracer**, including DHCP and static IP configurations, switch and router setup, and connectivity testing.

## Project Overview
- Designed a functional LAN topology with multiple PCs and laptops.
- Configured **static and DHCP IP addressing**.
- Tested connectivity using **ping commands** between devices.
- Verified router and switch configurations.

## Tools & Equipment
- Cisco Packet Tracer
- Router, Switch, PCs, Laptops (simulated)
- IP addressing scheme: 172.50.1.0/16
